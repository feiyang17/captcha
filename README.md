### 简介

一种`简单`、`安全`、`有趣`的新型验证方式。

* **1**种破解方式就可以辨认**75%**的文字验证码；
* **10**个人中就有**7**个人对难以识别的验证图片感到麻烦；
* **100**条验证短信就有**5**条无法被成功接收；
* **1000**个人中就有**400**人抱怨电话验证不友好；

基于手机特性，我们提出一种简单摇一摇、点一点、划一划即可有效而又安全通过验证的新方案。
从此和“看不清、换一张”分手吧，`11月22日`验证时代2.0即将揭幕。
